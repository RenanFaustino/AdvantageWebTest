DESCRIÇÃO DA ESTORIA
	Como usuário, 
	Eu quero criar uma conta, 
	Para que eu possa usufruir todas as funcionalidades da app http://advantageonlineshopping.com/#/.
	
REGRA DE NEGOCIO
	->	Deve conseguir inserir dados do cadastro
	->	Deve verificar os dados inseridos pelo usuário
	->	Deve acessar o aplicativo com sucesso após o cadastro


REGRA DE TELA
		A aplicação deve:

		Apresentar formulário de login
		- Campo com descrição para inserir Username;
		- Campo com descrição para inserir e-mail;
		- Campo com descrição para inserir Password;
		- Campo com descrição para inserir CorfirmPassword;
		- Campo com descrição para inserir First Name;
		- Campo com descrição para inserir Last Name;
		- Campo com descrição para inserir Phone Number;
	    - Campo com descrição para inserir Country;
	    - Campo com descrição para inserir City;
	    - Campo com descrição para inserir Adress;
	    - Campo com descrição para inserir State/Province/Region;
	    - Campo com descrição para inserir Postal Code;
